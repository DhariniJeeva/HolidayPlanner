Holiday planner

Index.html - log in to the page using name and email address. 
if any stored data found, then messages will be displayed accordingly for the user.

page2.html - will show the stored data(cookies/localstorage) from the first page. asks the user to select options from the drop down list. 

used animation on the first image.

Based on the selection, destination is displayed as an image and details about the selection will be displayed next to it. 

each selection will have its respective destination.

there is also a feature which automatically sends the details to the user's email address(local storage/cookies) taken from first page.

At the end, if the user is given a feature to send all the displayed info to his phone number. 

Validation is done on phone number input.
